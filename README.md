# CONNECTING DOT

AI 기반 유휴자산 · 창업 실험 · 지역 활성화 플랫폼

## 소개

CONNECTING DOT은  
유휴 공간·공장·장비·푸드트럭 등의 자산을  
창업가·학생·소상공인·지역 프로젝트와 연결하는  
AI 기반 창업 실험 플랫폼입니다.

단순 공간 중개가 아닌,

- 창업 실험
- AI 매칭
- 지역 활성화
- 청년 창업 지원
- 데이터 기반 성공 분석

을 통합하여  
누구나 작은 도전을 시작할 수 있는 환경을 만드는 것을 목표로 합니다.

---

## 주요 기능

### 공간 등록
유휴 점포, 공장, 장비, 이동형 공간 등록

### AI 매칭
업종·예산·지역·운영기간 기반 추천

### 창업 실험
단기 팝업 및 테스트 운영 지원

### 데이터 분석
운영성과 및 성공패턴 분석

### 지역 활성화
지역 소상공인 및 청년 창업 연계
